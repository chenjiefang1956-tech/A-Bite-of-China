## 项目定位
- 项目名称：A Bite of China，中国美食地图网页项目。
- 灵感来源：《舌尖上的中国》节目。
- 产品目标：用一张具有沉浸感的中国地图承载地域、美食、城市、方言与影像式叙事，让用户通过点击省份探索当地代表美食。
- 首页第一视觉必须是中国地图，地图是页面主角，不能被营销式首屏、装饰性大标题、卡片堆叠或无关插画抢占视觉中心。

## PRD

### 目标用户
- 对中国地域美食、纪录片式叙事、地方文化和旅行探索感兴趣的用户。
- 访问场景以桌面浏览为主，同时需要兼顾移动端基础可用性。

### 核心体验
1. 首页显示一张中国地图，地图占据页面主视觉。
2. 用户点击某个省份后，地图平滑推进，将该省份居中并放大。
3. 当前省份高亮显示，其他省份弱化处理。
4. 省份放大后，同步显示该省份相关城市名称。
5. 点击省份时播放一段该省份代表方言音频。
6. 当前省份下方或前景出现“味觉胶片”美食轮播。
7. 轮播图默认自动慢速滚动。
8. 用户按住鼠标左右拖动时，轮播必须跟随鼠标方向移动。
9. 拖动速度越快，轮播惯性越强；松手后逐渐减速，再恢复自动滚动。
10. 点击某道美食卡片后，弹出详情窗口展示该美食信息。

### 地图交互要求
- 中国地图必须是可交互地图，不得用静态图片假装点击省份，除非仅作为加载失败的降级展示。
- 省份点击后需要有连续、平滑的推进动画，避免瞬间跳变。
- 当前省份需要明确高亮；未选中省份需要弱化但仍保持地图整体轮廓可识别。
- 放大状态下必须保留返回全国视图的能力。
- 城市名称应与当前省份绑定展示，不得显示与当前省份无关的城市。
- 地图视图、当前省份、城市标注和下方内容必须保持状态同步。

### 方言音频要求
- 每个配置了方言音频的省份，在点击省份时播放对应音频。
- 切换省份时，上一段音频必须停止或淡出，避免多段音频叠加。
- 浏览器自动播放受限时，必须优雅处理：用户点击省份已经属于交互，可在该交互中触发播放；播放失败时不应导致页面报错。
- 音频素材必须有清晰来源或项目内说明，不能使用版权状态不明的节目原声作为默认素材。

### 味觉胶片轮播要求
- “味觉胶片”应呈现为横向美食卡片轮播，视觉上有胶片、影像或纪录片质感。
- 轮播默认自动慢速滚动，速度要稳定、克制，不影响用户阅读。
- 鼠标按住并左右拖动时，轮播位置必须实时跟随拖动方向。
- 松手时根据拖动速度产生惯性，速度越快惯性越明显。
- 惯性滚动需要逐渐减速，停止或接近停止后恢复默认自动滚动。
- 拖拽过程中应避免误触点击；只有非拖拽点击才打开详情弹窗。
- 移动端应支持触摸拖动，交互原则与鼠标拖动一致。

### 美食详情弹窗要求
- 点击美食卡片后打开详情窗口。
- 详情窗口至少展示：美食名称、所属省份或城市、图片、简介。
- 弹窗必须支持关闭操作，包括显式关闭按钮和点击遮罩或 Esc 键关闭。
- 弹窗打开时应避免背景内容无意义滚动；关闭后恢复之前的地图和轮播状态。

## 项目实现规则

### 视觉与体验
- 页面应服务于“地图探索 + 纪录片式美食发现”的体验，不做普通餐饮营销落地页。
- 避免大面积单一色系，尤其避免整页只由深蓝、紫蓝渐变、米色或棕橙色构成。
- UI 控件应紧凑、清晰、可扫描；卡片圆角不超过 8px，除非后续设计系统明确要求。
- 不得使用无意义的装饰光斑、渐变球、纯氛围背景替代真实内容。
- 文案不能遮挡地图、省份名称、城市标注、轮播卡片或弹窗内容。

### 数据与素材
- 省份、城市、美食、图片、音频应通过结构化数据维护，避免把内容散落硬编码在多个组件中。
- 每个省份数据至少应能关联：省份名称、城市列表、代表美食列表、可选方言音频。
- 图片应尽量使用真实美食或地域相关素材；不能用与内容无关的通用风景图糊弄主体验。
- 对缺失图片或音频的省份必须有降级方案，不得导致页面崩溃。

### 工程约束
- 优先沿用项目已有框架、组件风格和数据组织方式；没有既有实现时，采用简单、清晰、可维护的结构。
- 地图、轮播、音频、详情弹窗应拆分为边界清晰的模块，不把所有逻辑堆在单个文件中。
- 交互状态应有单一可信来源，避免地图选中状态、音频状态、轮播内容和弹窗内容互相不同步。
- 对复杂动画或拖拽惯性逻辑，应优先使用成熟浏览器 API 或可靠的小型实现，不引入不必要的大型依赖。
- 任何新增依赖都必须服务于核心体验，并在提交说明中解释原因。

### 性能与可访问性
- 首页地图和首屏交互必须优先加载；大图、音频和非当前省份资源应延迟加载或按需加载。
- 地图推进、轮播拖拽和惯性动画应保持流畅，避免因频繁重排造成卡顿。
- 所有可点击元素必须有可识别的焦点状态和键盘可达性。
- 图片必须有合适的替代文本；音频触发失败时应有非阻塞反馈。
- 弹窗应具备基本可访问性语义，包括标题、关闭方式和焦点管理。

### 验收标准
- 打开首页后，中国地图是明确主角。
- 点击省份后，地图平滑居中放大，当前省份高亮，其他省份弱化。
- 放大后只显示当前省份相关城市名称。
- 点击省份会尝试播放对应方言音频，切换省份不会产生音频叠加。
- 当前省份对应的“味觉胶片”轮播出现并自动慢速滚动。
- 鼠标或触摸拖动轮播时，轮播跟手移动；松手后有与速度相关的惯性，并逐步恢复自动滚动。
- 点击美食卡片会打开详情弹窗，弹窗内容与所点美食一致并可关闭。
- 缺失单个省份图片或音频素材时，页面仍可继续使用。
